Урок 4. Побітові операції

Побітові логічні операції та операції зсуву.

Одиницею виміру пам яті комп'ютера є байт. Кожний байт складається із восьми бітів. У кожному біті можна розмістити одне з двох чисел: О та 1. У одному байті можна зобразити ціле число з діапазону від О до 255. Щоб занести значення деякого даного у пам'ять комп'ютера, необхідно це дане перетворити у двійковий код. Зазвичай відповідні програми це роблять автоматично . Однак інколи на практиці, наприклад, під час створення графічних зображень чи в задачах архівування даних, виникає потреба порівнювати не значення двох даних, а їхні біти . Для цього у мові С++ є побітові логічні операції (див. табл. 8): & (І), І (АБО), л (виключне АБО), - (НЕ).

Операція Назва Приклади
~ Побітова not ~j
& побітова and i & 128
^ побітова xor j ^ 12
| побітова or j | 64
<< зсув вліво j << 2
>> зсув вправо j >> 3

Побітові операції not, and, or виконуються над бітами за такими правилами як для логічних даних. Наведемо правила виконання операцій.
Операція and. Результатом операції AND (&) (логічне І) для двох бітів буде одиниця, якщо обидва біти дорівнюють одиниці, або нулю, якщо хоча б один з них дорівнює нулю:

& 1 0
1 1 0
0 0 0

Операція xor. Результатом операції XOR (^) (виключне АБО) для двох бітів буде одиниця, якщо обидва біти різні, і результат дорівнює нулю, якщо обидва біти одинакові:

| (xor) 1 0
1 0 1
0 1 0

Результатом операції OR (|) (логічне АБО) для двох бітів буде одиниця, якщо хоча б один з них дорівнює одиниці, якщо обидва біти дорівнюють нулю, то результат дорівнює нулю.

| (or) 1 0
1 1 1
0 1 0

Операції зсуву виконуються за такими правилами.

1. x<<n
2. При зсуві вліво ліві n бітів зникають, праві n бітів заповнюються нулями.
3. x>>n
4. При зсуві вправо праві n бітів зникають, ліві n бітів заповнюються знаковим бітом: нулями - для числа дотатнього, одиницями - для числа від'ємного.

Приклад програми

Нехай b1 =00110011, b2=11110101.
Тоді b1 & b2 = 00110001 , b1 І b2 = 11110111 , b1 ^ b2 = 1100011 , ~ b1 = 110011 00.
Кожний байт можна зобразити у вигляді послідовності з восьми нулів та одиниць. Для того , щоб зсунути біти послідовності вправо чи вліво на деяку фіксовану кількість, використовують спеціальні операції зсуву >> та << .
Під час зсуву на порожні місця дописують нулі. Наприклад, нехай b = 00100101, с = 10110010, тоді під час виконання команд:

b = b << 2;
с = с >> 1;

змінна b набуде значення 10010100, а с- 01011001. Увага! Під час зсуву цифр цілого числа на n позицій вліво відбувається множення цього числа на 2n, а на n позицій вправо - ділення на 2n. Увага! Подібно до команд присвоєння 3 арифметичними операціями можна використовувати команди присвоєння з логічними побітовими операціями: &=, І=, л= та команди присвоєння з побітовим зсувом : >>=, <<+.

Команда розгалуження if (якщо)

Команда розгалуження if має дві форми: повну та коротку. Повна така:

if (<логічний вираз>) <команда 1 > ; else <команда 2> ;

Дія команди. Обчислюється значення логічного. виразу . Якщо це значення істинне, то виконується команда 1, у протилежному випадку - команда 2. Команда 1 та команда 2 можуть бути порожніми, простими або складеними.

if (х <= 5) z = ехр(х) ;
else z = ++х;

Коротка команда розгалуження if має вигляд

if (<логічний вираз>) <команда 1> ;

Дія команди. Обчислюється значення логічного виразу. Якщо воно істинне, то виконується команда 1, інакше виконується команда, яка записана після команди if.
